# ❤️ Heart Failure Prediction

A Machine Learning project that predicts the likelihood of heart failure based on a patient's medical information. This system helps identify individuals at risk of heart disease by analyzing various health parameters using predictive machine learning models.

---

## 📌 Overview

Heart disease is one of the leading causes of death worldwide. Early prediction can help doctors provide timely treatment and improve patient outcomes. This project uses Machine Learning algorithms to predict whether a patient is likely to develop heart failure based on medical attributes.

---

## 🚀 Features

- Predicts the risk of heart failure.
- Data preprocessing and cleaning.
- Handles missing values (if any).
- Feature selection and visualization.
- Machine Learning model training.
- Model evaluation using performance metrics.
- Predicts heart failure for new patient data.

## 📂 Dataset

The project uses a Heart Failure Prediction dataset containing patient health information such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- ST Depression (Oldpeak)
- ST Slope
- Heart Disease (Target)

---

## 🔄 Project Workflow

1. Load the dataset.
